Personal portfolio site for Hariraj Venkatesan, built with React and Vite and published through GitHub Pages.

## Tech Stack

- React 19
- Vite 8
- React Router
- Bootstrap and Bootstrap Icons
- Vitest with Testing Library

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm start
```

Build the production site:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Testing

Run the test suite:

```bash
npm test
```

Use watch mode while developing:

```bash
npm run test:watch
```
